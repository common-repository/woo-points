== Description ==

Increase your sales with WooPoints, the definitive Woocommerce Points and Rewards System.
Your customers will receive as reward points that can be redeemed in future purchases.

## My Points page ##
From 'My Points' page your customer can see their points stats, and request their coupons.
You can use the available shortcodes to customize the page.

## Shortcodes ##
**[woopoints_user_points]** Display the total user points.

**[woopoints_user_points_details]** Display a table with the user points detail.

## Dashboard ##
As administrator, you can manager the points generated, edit them or create points manually from the dashboard.

== Installation ==

1. Upload or extract the "woo-points" folder to your site's "/wp-content/plugins/" directory. You can also use the "Add new" option found in the "Plugins" menu in WordPress.
2. Enable the plugin from the "Plugins" menu in WordPress.
